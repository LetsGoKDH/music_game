# 엠준위 Music Quiz

엠준위 행사 진행자가 쓰는 음악 퀴즈 웹앱입니다.

## 접속 주소

- https://letsgokdh.github.io/music_game/

## 게임 개요

- 팀전 음악 퀴즈 진행용 페이지
- 라운드 시간: 3분
- 팀 수: 3팀 또는 4팀 선택
- 점수 방식: 정답 처리 시 현재 차례 팀 +1점
- 패스: 게임당 3회 (같은 팀 유지, 곡만 스킵)

## 진행 방식

1. 시작 화면에서 팀 수를 고르고 팀 이름을 입력합니다.
2. `게임 시작`을 누르면 타이머가 시작되고 곡이 재생됩니다.
3. 현재 차례 팀이 곡 제목/가수를 맞히면 `정답 처리`를 누릅니다.
4. 정답을 공개할 때는 `정답 보기`를 사용합니다.
5. 넘기려면 `Pass`를 누르면 팀은 유지된 채 다음 곡으로 넘어갑니다.
6. 시간이 끝나거나 `게임 종료`를 누르면 결과 화면에서 순위를 확인합니다.

## 팀 전환 규칙

- 팀 전환은 `정답 처리`로 점수 기록이 발생할 때만 진행됩니다.
- `Pass`는 팀 전환 없이 곡만 변경됩니다.

## 화면 설명

- YouTube 플레이어: 현재 문제 곡 재생
- 진행자 확인용 현재 곡: 진행자만 보는 제목/가수 표시 영역
- 팀 점수판: 팀별 현재 점수와 현재 차례 하이라이트
- 결과 화면: 최종 점수 기준 순위 정렬

## 데이터/동작 특징

- `songs.js`에 등록된 곡에서 랜덤 출제
- 이번 세션에서 재생한 곡 인덱스는 브라우저 `localStorage`에 저장
- `곡 목록 리셋` 버튼으로 누적 재생 기록 초기화 가능
- 곡 시작 전 광고/로딩 구간에는 타이머를 자동 일시정지하고 음소거 처리
- YouTube 임베드 제한 영상(100/101/150)은 자동으로 다음 곡으로 이동

## 사용 환경

- 권장: GitHub Pages 주소에서 접속 (`https://letsgokdh.github.io/music_game/`)
- 로컬 확인: `serve.bat` 실행 후 `http://localhost:8000`
- `file://`로 직접 열면 YouTube 재생이 제한될 수 있음
