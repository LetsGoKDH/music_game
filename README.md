# 엠준위 Music Quiz

행사 진행자가 바로 틀어놓고 쓰는 음악 퀴즈 페이지입니다.

## 어디서 실행하면 되나

- 가장 편한 방법: GitHub Pages에 올려서 휴대폰으로 접속
- 로컬 확인: `serve.bat` 실행 후 `http://localhost:8000`

## GitHub Pages 배포 순서

1. 이 폴더를 GitHub 저장소에 올립니다.
2. 저장소 `Settings > Pages`로 들어갑니다.
3. `Deploy from a branch`를 선택합니다.
4. 브랜치(`main`)와 폴더(`/ (root)`)를 선택하고 저장합니다.
5. 생성된 배포 주소로 접속합니다.

## 진행할 때 참고

- 게임 화면에서 YouTube 영상이 보이고, 현재 곡 제목/가수가 진행자용으로 같이 표시됩니다.
- `정답 보기`: 곡 제목/가수 공개
- `정답 처리`: 현재 팀 +1점 후 다음 팀으로 진행
- `Pass`: 점수 없이 다음 팀으로 넘김(3회 제한)

## 주의 사항

- 모바일에서 자동재생이 막히면 플레이어 재생 버튼을 한 번 눌러주세요.
- 일부 영상은 임베드가 막혀 있을 수 있어서 자동으로 다음 곡으로 넘어갑니다.
